JavaScript

  • script.js file is linked within the body element.
  • A variable is a named container that allows us to store data in our code.
  • To declare a variable, we use the var keyword
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops are used to control the order in which code is executed, or if it is even executed at all.
  • An if statement is a way of introducing decision-making into our code.
  • The if statement will only execute if the statement meets a condition, or is true
  • A value is considered falsy if it is one of the following values: 0, -0, On, " ", null, undefined, NaN, or a Boolean data type with the value of false.
  • A truthy is all other values, including a Boolean with the value of true
  • An array is a single variable that is used to hold a group of data.
  • An index is the location of the data item in the array, it starts counting at 0 and goes from there. [0, 1, 2, 3]
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A for loop contains the following three important statements: Starting point for our loop, Condition, inderate.
  • Indices follow a predictable pattern, it increasses by 1 each time. x++ === x+1
